# TrackCMS
目标：精简、小巧、快速的CMS系统。

技术架构：php+SQLite/Mysql

包含功能：文件管理、数据采集、Memcache整合、用户管理等

运行平台：常规PHP平台，SAE、BAE等云平台

作者博客：http://fengyu.name
